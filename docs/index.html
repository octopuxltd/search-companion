Search companion

A Firefox sidebar that opens alongside your Google search results — history, suggestions, and stuff from your browser, all in one panel.

Install for Firefox
Version 1.0.0 · Signed by Mozilla

What it does

  • Pinnable toolbar button to toggle the sidebar.
  • Search box that runs Google searches in the active tab.
  • Tabs for Current search and Saved searches.
  • Sections for search history, suggested next searches, and items from Firefox.

Install

  1. Open this page in Firefox.
  2. Click Install for Firefox above.
  3. Approve the prompt that appears.
